This Programming assignment is based on Python3.7.4
(The test transmission data is encoded in "utf-8")
(Blank "\x00" bytes is used, therefore the content of data is not important)

Operation:
Firstly, open windows command line and run PingServer.py:
C:\path\py PingServer.py 1500 (any port number greater than 1024 is acceptable)

Then, open a new command line window and run PingClient.py:
C:\path\py PingClient.py 127.0.0.1 1500
